# AlphaFold Protein Structure Database Dataset

## Overview

AlphaFold is a database of highly accurate protein structure predictions generated by DeepMind's AlphaFold system. Provides 3D structure predictions with per-residue confidence estimates (pLDDT scores) for proteins across the proteome. Contains 214+ million structure predictions covering UniProt sequences. Essential for structural biology, protein function prediction, drug discovery, and understanding protein mechanisms.

**Source**: AlphaFold Database (EMBL-EBI FTP)
**Data Type**: AI-predicted 3D protein structures with confidence scores stored as attributes on UniProt entries

## Integration Architecture

### Storage Model

**Primary Entries**:
- UniProt IDs (e.g., `A0A009IHW8`) serve as primary keys
- AlphaFold data stored as **attributes on UniProt entries**

**Searchable Text Links**:
- AlphaFold model IDs (e.g., `AF-A0A009IHW8-F1`) indexed as keywords
- Enables lookup by both UniProt ID and model ID

**Attributes Stored** (protobuf AlphafoldAttr):
- `global_metric`: Average pLDDT score (0-100, overall structure quality)
- `fraction_pldddt_very_high`: Fraction with pLDDT > 90
- `fraction_pldddt_confident`: Fraction with pLDDT 70-90
- `fraction_pldddt_low`: Fraction with pLDDT 50-70
- `fraction_pldddt_very_low`: Fraction with pLDDT < 50
- `model_entity_id`: AlphaFold model identifier (e.g., AF-A0A009IHW8-F1)
- `gene`: Gene symbol (when available)

**Cross-References**:
- **Bidirectional**: UniProt (dataset 1) ↔ AlphaFold (dataset 30)
- **Structure databases**: PDB (for experimentally solved structures)
- **Functional annotations**: Via UniProt cross-references

### Special Features

**pLDDT Confidence Scoring**:
- Per-residue confidence estimates from AlphaFold predictions
- Four confidence categories with fractions summing to 1.0
- Global metric provides overall structure quality assessment

**Dual Lookup Methods**:
- Query by UniProt ID → Returns AlphaFold attributes
- Query by model ID (e.g., AF-A0A009IHW8-F1) → Resolves to UniProt entry

**Swiss-Prot Subset**:
- Test data uses SwissProt subset (~550K reviewed proteins)
- Full database: 214M+ predictions across all UniProt
- Streaming tar.gz processing (no full extraction needed)

**Structure File References**:
- PDB and mmCIF format URLs provided
- Predicted Aligned Error (PAE) images available
- Confidence score JSON files accessible

## Use Cases

**1. Structure-Based Function Prediction**
```
Query: UniProt ID → AlphaFold structure + pLDDT scores → Identify functional domains
Use: Predict protein function from structural features
```

**2. Drug Target Analysis**
```
Query: Target protein → AlphaFold structure → Identify binding pockets
Use: Structure-based drug design and virtual screening
```

**3. Protein Engineering**
```
Query: Protein → Confidence scores → Identify flexible/rigid regions
Use: Guide mutagenesis and protein design experiments
```

**4. Structural Coverage Assessment**
```
Query: Protein family → Check AlphaFold confidence → Prioritize experimental structures
Use: Decide which proteins need experimental structure determination
```

**5. Model Quality Filtering**
```
Query: Proteins with global_metric > 70 → High-confidence structures only
Use: Filter for reliable structural models in large-scale studies
```

**6. Comparative Modeling**
```
Query: Related proteins → AlphaFold structures → Structural alignment
Use: Understand evolutionary relationships and functional divergence
```

## pLDDT Confidence Interpretation

| pLDDT Range | Confidence Level | Interpretation |
|-------------|------------------|----------------|
| **> 90** | Very High | High accuracy, well-ordered regions |
| **70-90** | Confident | Generally good backbone prediction |
| **50-70** | Low | Low confidence, potential disorder |
| **< 50** | Very Low | Should not be interpreted, likely disordered |

**Global Metric**: Average pLDDT across all residues (0-100 scale)

## Known Limitations

**Version Tracking**:
- Currently stores latest version only (v6)
- Historical versions not tracked

**Fragment Structures**:
- Some proteins have multiple fragments (F1, F2, etc.)
- Each fragment stored as separate model

**Isoforms**:
- AlphaFold typically models canonical sequence only
- Alternative isoforms may lack predictions

**Confidence Interpretation**:
- pLDDT measures local accuracy, not global fold accuracy
- High pLDDT regions can still have incorrect relative orientations
- Does not guarantee biological relevance

**PAE Data**:
- Predicted Aligned Error matrices not stored
- Only summary confidence metrics retained
- PAE images/JSON available via external URLs

**Experimental Validation**:
- Predictions are computational, not experimentally validated
- Should be verified with experimental data when available
